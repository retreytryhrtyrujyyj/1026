# 个人技术博客

## 项目简介
个人技术博客分前台和后台系统。前台系统查看博客，搜索博客，评论博客等。后台系统发布博客，审核博客评论，修改用户信息，登录/注册。基于Java语言，SSM框架，Mysql数据库，JSP技术。

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=1026

## 技术介绍
Java Web 、JSP、SSM框架、Mysql数据库

## 开发工具
Eclipse或者MyEclipse或者IDEA Navicat

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
